Frontend-часть веб-приложения для работы с генеалогическим деревом династии Романовых.

Технологии:
- React (Vite)
- JavaScript (ES6+)
- React Router

##Реализованный функционал:
Навигация:
- Навигационное меню (Navbar)
- Переход между страницами:
 - Главная (/)
 - Поиск (/search)

Главная страницы (UC-01):
- Отображение статистики:
 - количество персон
 - средний возраст
 - период существования династии
- Список последних добавленных персон

Страница поиска (UC-02):
- Форма фильтрации (имя, фамилия, титул, пол, годы рождения и смерти)
- Отправка запроса к API
- Отображение результатов в таблице
- Сообщение “Ничего не найдено”

Карьточка персоны (UC-03):
- Открытие модального окна при клике на строку таблицы
- Закрытие (по кнопке, по клику вне окна)
- В текущей версии реализована как заглушка


Архитектура проекта:
```
src/
├── api/ # функции для работы с backend API
├── components/ # переиспользуемые компоненты
│ ├── Navbar.jsx
│ └── PersonModal.jsx
├── pages/ # страницы приложения
│ ├── Home.jsx
│ └── Search.jsx
├── router/ # настройка маршрутов
└── style.css # глобальные стили
```

Работа с API
Все запросы вынесены в отдельный модуль:

GET /api/stats
GET /api/persons/search
GET /api/persons/recent

Используется fetch для получения данных.


Запуск проекта:

```
Установить зависимости:
npm install
Запустить приложение:
npm run dev
Открыть в браузере:
http://localhost:5173
```

Для корректной работы требуется запущенный backend
