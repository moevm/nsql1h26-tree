Backend-часть для приложения, визуализирующего генеалогическое древо династии Романовых.

Технологии:
- Python 3.12
- FastAPI
- Neo4j
- Docker & Docker Compose

Для запуска БД необходимо наличие файла `.env` в директории `backend`, 
в котором находятся данные для подключения к Neo4j в следующем формате: 
```
NEO4J_URI=bolt://db:7687
NEO4J_USER=some_name
NEO4J_PASSWORD=some_password
```

Для запуска приложения необходимо, находясь в директории `backend` выполнить следующую команду:
```
docker compose build --no-cache && docker compose up
```
После завершения работы с приложением необходимо, находясь в директории `backend` выполнить следующую команду:
```
docker compose down
```

После запуска будут доступны следующие интерфейсы: 
- Swagger UI `http://localhost:8000/docs`
На данном сайте можно протестировать некоторые реализованные запросы к БД:

- Neo4j Browser `http://localhost:7474`
На данном сайте запущена Neo4j, необходимо при входе указать данные для подключения: имя пользователя и пароль, 
которые указаны в вашем файле `.env`. После авторизации можно выполнять запросы к БД на `Cypher`.

Основные функции: 
- `GET /api/stats/` - Статистика по династии (кол-во персон, связей, средняя продолжительность жизни)
- `GET /api/persons/search` - Поиск персон с фильтрацией (по имени, фамилии, датам рождения и смерти, полу)
- `GET /api/persons/{id}` - Карточка персоны со всеми родственными связями
- `POST /api/persons` - Добавление новой персоны со связями
